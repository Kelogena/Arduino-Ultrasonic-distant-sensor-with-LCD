# 📡 Arduino Ultrasonic Distance Detector with LCD 1602 & Buzzer

A real-time distance measurement system built using **Arduino Uno**, **HC-SR04 ultrasonic sensor**, **16x2 LCD display**, and a **piezo buzzer**. The system measures distance to objects and provides both visual feedback (LCD) and audio alerts (buzzer intensity changes depending on distance).

---

## 📋 Features

- Real-time distance measurement (2 cm – 400 cm)
- LCD 1602 display output
- Audio feedback using buzzer (distance-based alert levels)
- Serial Monitor debugging support
- Stable readings with timeout protection
- Beginner-friendly Arduino robotics project

---

---

## 📸 Hardware Setup

The assembled prototype includes:

- Arduino Uno board
- HC-SR04 ultrasonic sensor facing forward
- LCD 1602 display for distance output
- Piezo buzzer for sound alerts
- Potentiometer for LCD contrast adjustment
